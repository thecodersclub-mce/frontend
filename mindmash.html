21. Task 1: Basic HTML Structure (The Foundation)

Goal: Create the standard HTML boilerplate and display a simple heading and paragraph.

Requirements:

  1. Create a complete HTML document with the necessary tags (<!DOCTYPE html>, <html>, <head>, <body>).
  2. Inside the <head>, include a <title> tag with the text "My First Page".
  3. Inside the <body>, add an <h1> heading that says "HTML Starter".
  4. Below the heading, add a paragraph <p> that says "This is the start of my web development journey.".

22. Task 2: Intermediate HTML & CSS (Styling and Links)

Goal: Structure content, apply a class, and use CSS to style it.

Requirements:

  • HTML: Add a <div> element with the class card. Inside the div, include an <h2> heading: "CSS Card Design". Add a link <a> with the text "Visit Google" that points to https://www.google.com.
  • CSS: (Imagine this is in a separate <style> block or file) Style the .card class with border: 1px solid #ccc; and padding: 20px;. Change the background color of the <body> to #f4f4f4.
---

23. Task 3: JavaScript DOM Manipulation (Interactivity)

Goal: Use JavaScript to change content on the page when a button is clicked.

Requirements:

  • HTML: Add a button <button> with the ID change-btn and the text "Change Text". Add a paragraph <p> with the ID output-text and initial text "The text will change soon.".
  • JavaScript: Select both the button and the paragraph using their IDs. Add an event listener to the button that responds to a click. When clicked, the function should change the inner text of the paragraph (output-text) to "The text has been updated by JavaScript!".
---

24. Task 4: Functional React Component (JSX and Props)

Goal: Write a basic functional React component that accepts and displays dynamic data using props.

Requirements:

  1. Define a functional component named GreetingCard that takes a single name prop.
  2. The component must return JSX that displays an <h2> heading.
  3. The heading should dynamically display the text: "Hello, [Name]! Welcome to React." (where [Name] is the value of the prop).
  4. Assume React is already imported.

25. Task 5: React Component with useState (Basic State)

Goal: Implement the useState hook to make a simple counter component.

Requirements:

  1. Define a functional component named ClickCounter.
  2. Use the useState hook to manage a state variable named count, initialized to 0.
  3. The component must display the current count in a paragraph: "You clicked [Count] times."
  4. Include a button that, when clicked, uses the updater function to increase the count by 1.
  5. Assume useState is already imported from React.
