5. Task: Interactive CSS Toggling (Intermediate DOM)

Goal: Use a single button click to toggle a CSS class on the <body> element, switching the page between two distinct themes (simulating a dark/light mode switch).

Requirements:

  1. HTML: Create a complete HTML document structure. Inside the <body>, add a <button> with the ID theme-toggle and the text "Toggle Theme".


  2. CSS: (Imagine this is in a separate <style> block or file) Define the default light theme on the body selector: Set background-color: #ffffff; and text color: #000000;. Define a class selector named .dark-mode. Set its styles to the dark theme: background-color: #333333; and text color: #f0f0f0;.


  3. JavaScript: (Imagine this is in a separate <script> block or file) Select the button (theme-toggle) and the <body> element. Add an event listener to the button that responds to a click. The function should use the JavaScript classList.toggle() method on the <body> element to dynamically add or remove the .dark-mode class.
